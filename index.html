Soirée Silicium

Les cours en distanciel ont beaucoup dégradé la proximité des MMI. Afin de retrouver du lien entre étudiants, mais également avec la formation elle-même, nous avons créé la Soirée Silicium !

Nous invitons celles et ceux qui le souhaitent à se joindre à ce projet qui a pour but de redécouvrir en équipe les meilleurs aspects du DUT MMI !

La Soirée Silicium est un évènement de plusieurs heures durant lequel vous pourrez produire en groupe des créations originales et innovantes en mettant en œuvre les enseignements dispensés en MMI, qu'ils soient artistiques ou informatiques.

  • 10 février, de 20h à 3h via Discord
  • Réaliser des projets
  • Les meilleurs aspects du DUT MMI

Les challenges

Les challenges, c'est comme des petits projets mais sans restriction, histoire de donner des idées aux personnes qui n'ont pas de projet. En fin de soirée on pourra comparer ce que les équipes ont fait pour un même challenge. Vous êtes complètement libres de faire le nombre de challenge que vous voulez et de les choisir à votre guise dans la liste ci-dessous. Vous avez déjà un projet d’équipes mais un ou plusieurs challenges vous intéressent aussi ? Rien ne vous empêche de les faire en plus !

Programmation

Le Manoir d’Houdin-e

Créer un site web sous forme d’escape game/d’énigme

Captur The Flag sur le site www.soireesilicium.fr

Trouvez les drapeaux ! (les drapeaux sont des chaînes de caractère a des endroits cachés (côté client) il y a 4 drapeaux ! ex de drapeau : cfcd208495d565ef66e7dff9f98764da Compétences nécessaires : html + css + js

Nom de code A.P.I

Créer une api en node js ou php. L’api permettrait de donner un élément aléatoire d’une table dans une base de donnée (mysql pgsql ou mongodb) Compétence nécessaire : js ou php + sql ou nosql
